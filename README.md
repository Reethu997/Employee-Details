## Employee Directory
This project is an Employee Directory application built with React. It allows you to view, add, update, and delete employee records in a table format.

### Features
View a list of employees with their ID, name, age, date of birth, email, phone, salary, gender, and actions.
Sort the table by clicking on the Name column headers.
Search for employees by their name or ID.
Add a new employee by clicking the "Add Employee" button.
Edit an existing employee's information.
Delete an employee from the directory.
